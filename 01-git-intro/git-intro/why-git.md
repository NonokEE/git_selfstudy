# Git을 사용하는 이유

## 핵심 이유 
1. 변경 내용을 모두 기록하므로, 버전 관리에 용이하다!
2. 마찬가지로 변경 내용을 모두 기록하므로, git을 사용하고 commit하는 것 만으로 백업이 된다!
3. 협업에 용이하다!

## Git이 없다면 어떤 불편함이 생길까?
