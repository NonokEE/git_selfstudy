# Git & GitHub Self-Study

Git과 GitHub를 단계적으로 학습하고, 주제별 실습 과정과 결과를 기록하는 저장소입니다.

## 학습 목표

- Git의 버전 관리 흐름과 핵심 명령어를 이해합니다.
- 브랜치, 병합, 충돌 해결, 이력 복구를 직접 실습합니다.
- GitHub 기반 협업 흐름과 Pull Request를 익힙니다.

## 학습 순서

1. Git 입문
2. Git 초급
3. Git 중급
4. GitHub 입문
5. GitHub 초급
6. Git 고급
7. GitHub 중급
8. GitHub 고급

## 저장소 구조

- `Learning_progress.md`: 전체 학습 진행 상황
- `git-intro/`: Git 입문 실습 기록
- `git-branch-practice/`: 브랜치 관련 실습 기록
- `git-merge-practice/`: 병합 관련 실습 기록
- `git-rebase-practice/`: rebase 관련 실습 기록
- 그 외 주제별 폴더: 각 Git/GitHub 학습 실습 기록

## 학습 진행 상황

전체 학습 진행 상황은 [Learning_progress.md](Learning_progress.md)에서 확인할 수 있습니다.

## 작성 원칙

- 주제별 실습 과정과 결과를 Markdown 문서로 기록합니다.
- 하나의 학습 목적에 맞춰 커밋을 작성합니다.
- 실습 전후의 차이와 배운 점을 함께 남깁니다.
- 공유된 이력은 임의로 수정하지 않고, 필요한 경우 안전한 Git 명령을 사용합니다.
